## ⚡ Overview

An ultra-modern, IDE-inspired personal portfolio built with **Next.js 15** — themed **"The Terminal Horizon"**. The entire UI is designed to feel like a high-end VS Code environment: syntax-highlighted code lines scroll as a living background, every section lives inside a code editor window, and the navigation mimics VS Code file tabs. Powered by Lenis smooth scroll + GSAP for 60fps animations.

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Jhnllyd11/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## 📜 Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |

---

## 🔐 Environment Variables

Create a `.env` file in the root:

```env
NEXT_PUBLIC_EMAILJS_SERVICE_ID=<your_service_id>
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=<your_template_id>
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=<your_public_key>
```

> ⚠️ Never commit your `.env` file — it's already in `.gitignore`.
